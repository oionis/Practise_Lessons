Inner Class (внутренний класс)

Внутренние классы, тесно связаны с внешним (тот внутри которого он создан). Оба класса имеют доступ ко всем полям другого класса, даже private.
Экземпляр внутреннего класа не может быть создан внутри статического метода (на пример Main). Для Inner класса нужен какой-то уже созданный объект класса, к кторому он сможет "привязаться".
Без экземпляра внешнего класса, мы не сможем создать экземпляр внутреннего класса.

Внутренний класс на уровне метода. Очень ограничен в функционале. Экземпляр класса можно создать внутри метода, передать наружу экземпляр класса - нельзя.
Методы внутреннего класса можно вызывать только внутри метода, в котором этот класс создан.
